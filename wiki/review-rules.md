## 📋 **Checklist ก่อนส่งให้เหน่งรีวิว**

### **1. ตรวจสอบภาษา:**

```
✅ บทความเป็นภาษาไทย 100%
✅ ไม่มีภาษาอื่นหลุด (จีน, อังกฤษ, เวียดนาม, ฯลฯ)
✅ คำศัพท์เทคนิคมีคำแปลหรือคำอธิบาย
```

**คำสั่งตรวจสอบ:**
```bash
# ค้นหาคำจีนที่อาจหลุด
grep -E "你将 | 你会 | 中文 | 学习" content/posts/[article]/index.md

# ค้นหาคำอังกฤษที่อาจหลุด (ยกเว้น technical terms)
grep -E "[A-Z]{3,}" content/posts/[article]/index.md
```

---

### **2. ตรวจสอบชื่อแบรนด์/ผลิตภัณฑ์:**

```
✅ Twitter → X (อัพเดท 2026)
✅ Facebook → Facebook (ถูกต้อง)
✅ LINE → LINE (ถูกต้อง)
✅ Telegram → Telegram (ถูกต้อง)
✅ LinkedIn → LinkedIn (ถูกต้อง)
```

**คำสั่งตรวจสอบ:**
```bash
# ค้นหา Twitter ที่อาจยังไม่ได้แก้
grep -rn "Twitter" content/posts/ --include="*.md" | grep -v "twitter.com"
```

---

### **3. ตรวจสอบลิงก์:**

```
✅ ลิงก์ไปบทความอื่นถูกต้อง
✅ ลิงก์ภายนอกใช้งานได้
✅ Internal linking ครบถ้วน
```

---

### **4. ตรวจสอบรูป:**

```
✅ มีรูปปก (cover.jpg)
✅ มีรูปประกอบ 1-2 รูป
✅ รูปไม่ซ้ำกัน (เช็ค MD5)
✅ ใช้ relative path
```

**คำสั่งตรวจสอบ:**
```bash
# เช็ค MD5 รูปไม่ซ้ำ
cd content/posts/[article]/
md5sum *.jpg | sort | uniq -d -w32

# ต้องไม่มี output = ไม่มีรูปซ้ำ
```

---

### **5. ตรวจสอบ Emoji:**

```
✅ ใช้ emoji พอประมาณ (5-10 ตัว)
✅ ไม่เยอะเกินไปจนรก
✅ ใช้เน้นหัวข้อสำคัญ
```

---

### **6. ตรวจสอบโครงสร้าง:**

```
✅ มีสารบัญ (สำหรับบทความยาว)
✅ มีหัวข้อชัดเจน (H1, H2, H3)
✅ มี Internal Links
✅ มี CTA ท้ายบทความ
✅ มีส่วนอ้างอิง (ถ้ามี)
✅ มี "ตอนต่อไป" (สำหรับซีรีส์)
```

---

### **7. ตรวจสอบ SEO & Social:**

```
✅ Meta Description ครบ
✅ OG Image ถูกต้อง
✅ OG Title/Description
✅ Share Buttons พร้อม
✅ Share Buttons ใช้ชื่อถูกต้อง (X ไม่ใช่ Twitter)
```
