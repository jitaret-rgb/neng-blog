## 🤔 **Prompt Engineering คืออะไร?**

![Prompt Engineering Concept](image1.jpg)
*ภาพ: แนวคิด Prompt Engineering - ภาพประกอบจาก Unsplash*

### **คำนิยามง่ายๆ:**

```
Prompt Engineering = "ศิลปะการสั่ง AI ให้ได้ผลลัพธ์ที่ต้องการ"
```

**เปรียบเทียบ:**
```
❌ สั่งมั่วๆ → AI งง → ผลลัพธ์แย่
✅ สั่งชัดเจน → AI เข้าใจ → ผลลัพธ์ดี
```

---

### **Prompt คืออะไร?**

```
Prompt = "คำสั่ง" ที่เราส่งให้ AI
```

**ตัวอย่าง:**
```
❌ Prompt แย่: "เขียนโพสต์ Facebook"
✅ Prompt ดี: "เขียนโพสต์ Facebook แจ้งเตือนชาวบ้านเรื่องประชุม อบต. วันเสาร์นี้ 10 โมง"
```

---

## 🎯 **ทำไมต้องเรียน Prompt Engineering?**

### **ปัญหาที่พบบ่อย:**

```
❌ เขียน Prompt ไม่ชัดเจน
❌ AI ไม่เข้าใจสิ่งที่ต้องการ
❌ ได้ผลลัพธ์ไม่ตรงใจ
❌ ต้องแก้ซ้ำหลายรอบ
❌ เสียเวลา
```

---

### **ถ้าเขียน Prompt เป็น:**

```
✅ AI เข้าใจทันที
✅ ได้ผลลัพธ์ตรงใจ
✅ แก้แค่นิดหน่อย
✅ ประหยัดเวลา 80%
✅ ทำงานเร็วขึ้น 3 เท่า
```

---

### **สถิติที่น่าสนใจ:**

| เขียน Prompt | เวลาที่ใช้ | คุณภาพผลลัพธ์ |
|------------|-----------|--------------|
| **ไม่มีความรู้** | 30 นาที | 50% |
| **มีความรู้พื้นฐาน** | 10 นาที | 80% |
| **เชี่ยวชาญ** | 3 นาที | 95% |

**สรุป:** เรียน Prompt Engineering คุ้มค่า! ⏱️

## 📝 **โครงสร้าง Prompt ที่ดี**

### **สูตร 5 องค์ประกอบ:**

```
┌─────────────────────────────────────┐
│   Prompt ที่ดี = 5 องค์ประกอบ       │
├─────────────────────────────────────┤
│   1. บทบาท (Role)                   │
│   2. งาน (Task)                     │
│   3. รายละเอียด (Details)           │
│   4. รูปแบบ (Format)                │
│   5. ข้อจำกัด (Constraints)         │
└─────────────────────────────────────┘
```

---

### **1. บทบาท (Role)**

**กำหนดให้ AI เป็นใคร:**

```
✅ "คุณคือ Content Creator เชี่ยวชาญการพัฒนาชุมชน"
✅ "คุณคือนักเขียนคอนเทนต์มืออาชีพ"
✅ "คุณคือผู้ช่วยนักพัฒนาชุมชน"
```

**ผลลัพธ์:** AI จะตอบในบทบาทที่กำหนด

---

### **2. งาน (Task)**

**บอกว่าให้ทำอะไร:**

```
✅ "เขียนโพสต์ Facebook แจ้งเตือนชาวบ้าน"
✅ "สรุปการประชุม อบต."
✅ "เขียนโครงการของบประมาณ"
```

**ผลลัพธ์:** AI รู้ว่าต้องทำอะไร

---

### **3. รายละเอียด (Details)**

**ให้ข้อมูลครบถ้วน:**

```
✅ "ประชุมวันเสาร์ที่ 13 เม.ย. 69 เวลา 10:00 น."
✅ "ณ ศาลาประชาคม อบต.ตัวอย่าง"
✅ "วาระสำคัญ: อนุมัติงบประมาณโครงการน้ำประปา"
```

**ผลลัพธ์:** AI มีข้อมูลครบ เขียนได้ถูกต้อง

---

### **4. รูปแบบ (Format)**

**กำหนดรูปแบบที่ต้องการ:**

```
✅ "เขียนความยาว 100-150 คำ"
✅ "ใช้ภาษาเป็นกันเอง"
✅ "มี emoji นิดหน่อย"
✅ "มี hashtag #ชุมชนเข้มแข็ง"
```

**ผลลัพธ์:** AI เขียนตรงตามรูปแบบที่ต้องการ

---

### **5. ข้อจำกัด (Constraints)**

**บอกว่าอะไรไม่ควรทำ:**

```
✅ "ไม่ต้องยาวเกินไป"
✅ "ไม่ต้องใช้ภาษาทางการ"
✅ "ไม่ต้องใส่ข้อมูลการเมือง"
```

**ผลลัพธ์:** AI ไม่ทำสิ่งที่เราไม่ต้องการ

---

## 🎨 **ตัวอย่าง Prompt เปรียบเทียบ**

### **❌ Prompt แย่:**

```
"เขียนโพสต์ Facebook"
```

**ผลลัพธ์:**
```
โพสต์ Facebook ทั่วไป ไม่เจาะจง
ไม่ได้ข้อมูลที่ต้องการ
ต้องแก้ใหม่
```

---

### **✅ Prompt ดี:**

```
คุณคือ Content Creator เชี่ยวชาญการพัฒนาชุมชน

จงเขียนโพสต์ Facebook แจ้งเตือนชาวบ้านเรื่องประชุม อบต.

รายละเอียด:
- วันเสาร์ที่ 13 เม.ย. 69
- เวลา 10:00 น.
- ณ ศาลาประชาคม อบต.ตัวอย่าง
- วาระ: อนุมัติงบประมาณโครงการน้ำประปา

รูปแบบ:
- ความยาว 100-150 คำ
- ภาษาเป็นกันเอง
- มี emoji นิดหน่อย
- มี hashtag #ชุมชนเข้มแข็ง #อบตตัวอย่าง

ข้อจำกัด:
- ไม่ต้องยาวเกินไป
- ไม่ต้องใช้ภาษาทางการ
```

**ผลลัพธ์:**
```
✅ โพสต์พร้อมใช้
✅ ข้อมูลครบถ้วน
✅ รูปแบบถูกต้อง
✅ ไม่ต้องแก้
```

## 💡 **เทคนิคเขียน Prompt สำหรับงานชุมชน**

### **เทคนิคที่ 1: ใช้ Template**

**สร้าง Template เก็บไว้:**

```
คุณคือ [บทบาท]

จง [งาน]

รายละเอียด:
- [รายละเอียดที่ 1]
- [รายละเอียดที่ 2]
- [รายละเอียดที่ 3]

รูปแบบ:
- [รูปแบบที่ต้องการ]

ข้อจำกัด:
- [สิ่งที่ควรหลีกเลี่ยง]
```

**ใช้ซ้ำได้:** แค่เปลี่ยนข้อมูลใน [ ]

---

### **เทคนิคที่ 2: ให้ตัวอย่าง**

**AI เรียนรู้จากตัวอย่าง:**

```
เขียนโพสต์คล้ายตัวอย่างนี้:

ตัวอย่าง:
"📢 ประกาศถึงชาวบ้านทุกคน!
ประชุม อบต. วันเสาร์นี้ 10 โมง
อย่าลืมมาเข้าร่วมกันนะครับ 🙏
#ชุมชนเข้มแข็ง"

โจทย์: เขียนประกาศเรื่องงานบุญบั้งไฟ
```

**ผลลัพธ์:** AI จะเขียนโทนคล้ายตัวอย่าง

---

### **เทคนิคที่ 3: แบ่งงานใหญ่เป็นงานย่อย**

**งานใหญ่ → แบ่งเป็นขั้นตอน:**

```
❌ "เขียนโครงการของบประมาณ" (ใหญ่ไป)

✅ แบ่งเป็น:
1. "เขียนหลักการและเหตุผล"
2. "เขียนวัตถุประสงค์"
3. "เขียนกิจกรรม"
4. "เขียนงบประมาณ"
5. "เขียนผลลัพธ์ที่คาดหวัง"
```

**ผลลัพธ์:** แต่ละส่วนคุณภาพดีกว่า

---

### **เทคนิคที่ 4: ให้ AI ถามกลับ**

**ถ้าข้อมูลไม่ครบ:**

```
"ถ้าข้อมูลไม่พอ ให้ถามผมเพิ่มก่อนเขียน"
```

**ผลลัพธ์:** AI จะถามข้อมูลที่ขาด

---

### **เทคนิคที่ 5: ให้ AI แก้เอง**

**สั่งให้ AI ตรวจสอบ:**

```
"เขียนเสร็จแล้ว ให้ตรวจสอบว่า:
1. ข้อมูลครบไหม
2. ภาษาถูกต้องไหม
3. มีความยาวตามที่กำหนดไหม

ถ้ามีข้อผิดพลาด ให้แก้เอง"
```

**ผลลัพธ์:** AI ตรวจสอบและแก้ให้เอง

## 📝 **ตัวอย่าง Prompt ใช้จริง**

![Prompt Examples](image2.jpg)
*ภาพ: ตัวอย่างการเขียน Prompt - ภาพประกอบจาก Unsplash*

### **1. เขียนโพสต์ Facebook**

```
คุณคือ Content Creator เชี่ยวชาญการพัฒนาชุมชน

จงเขียนโพสต์ Facebook แจ้งเตือนชาวบ้านเรื่องงานบุญ

รายละเอียด:
- งาน: งานบุญบั้งไฟประจำปี
- วัน: อาทิตย์ที่ 21 เม.ย. 69
- เวลา: 08:00 น. เป็นต้นไป
- สถานที่: วัดตัวอย่าง
- กิจกรรม: แห่บั้งไฟ, การแสดงพื้นบ้าน, ร้านค้าชุมชน

รูปแบบ:
- ความยาว 100-150 คำ
- ภาษาเป็นกันเอง
- มี emoji
- มี hashtag #งานบุญ #ชุมชนเข้มแข็ง

ข้อจำกัด:
- ไม่ต้องยาวเกินไป
- ไม่ต้องใช้ภาษาทางการ
```

---

### **2. สรุปการประชุม**

```
คุณคือผู้ช่วยนักพัฒนาชุมชน

จงสรุปการประชุม อบต.

ข้อมูลการประชุม:
- วันที่: 1 เม.ย. 69
- ผู้เข้าร่วม: นายก อบต., สมาชิก 15 คน, ชาวบ้าน 30 คน
- วาระ:
  1. อนุมัติงบประมาณโครงการน้ำประปา
  2. พิจารณาโครงการถนนคอนกรีต
  3. รับรองรายงานการประชุมครั้งก่อน

มติที่ประชุม:
1. อนุมัติโครงการน้ำประปา งบ 500,000 บาท
2. เลื่อนโครงการถนนออกไปก่อน
3. รับรองรายงาน

รูปแบบ:
- เขียนเป็นข้อๆ
- ภาษาทางการนิดหน่อย
- ความยาว 200-300 คำ
```

---

### **3. เขียนโครงการของบประมาณ**

```
คุณคือผู้เชี่ยวชาญการเขียนโครงการ

จงเขียนโครงการของบประมาณ

ข้อมูลโครงการ:
- ชื่อ: โครงการน้ำประปาชุมชน
- หน่วยงาน: อบต.ตัวอย่าง
- งบประมาณ: 500,000 บาท
- ระยะเวลาดำเนินการ: 6 เดือน
- กลุ่มเป้าหมาย: ชาวบ้าน 200 ครัวเรือน

เขียนเฉพาะ:
1. หลักการและเหตุผล (200 คำ)
2. วัตถุประสงค์ (3 ข้อ)
3. กิจกรรม (5 กิจกรรม)
4. งบประมาณ (แจกแจงรายละเอียด)
5. ผลลัพธ์ที่คาดหวัง

รูปแบบ:
- ภาษาทางการ
- เป็นทางการ
- ครบถ้วน
```

---

### **4. เขียนข่าวประชาสัมพันธ์**

```
คุณคือโฆษก อบต.

จงเขียนข่าวประชาสัมพันธ์

ข้อมูล:
- หัวข้อ: อบต.ตัวอย่าง ได้รับรางวัลชุมชนเข้มแข็งระดับประเทศ
- วันที่รับรางวัล: 5 เม.ย. 69
- ผู้รับ: นายก อบต.ตัวอย่าง
- จัดโดย: กรมการพัฒนาชุมชน
- สถานที่: อิมแพ็ค เมืองทองธานี

รูปแบบ:
- ความยาว 300-400 คำ
- ภาษาทางการ
- มีคำพูดนายก อบต.
- มีข้อมูลประโยชน์ที่ชาวบ้านได้รับ
```

---

### **5. เขียนคำพูดสำหรับนายก อบต.**

```
คุณคือ speechwriter (นักเขียนคำพูด)

จงเขียนคำพูดสำหรับนายก อบต.

โอกาส: เปิดงานบุญบั้งไฟ
ผู้ฟัง: ชาวบ้าน 500 คน
ความยาว: 5 นาที (ประมาณ 500-600 คำ)

เนื้อหา:
1. กล่าวต้อนรับ
2. ความสำคัญของงานบุญ
3. นโยบายการพัฒนาชุมชน
4. อวยพรให้งานสำเร็จ

รูปแบบ:
- ภาษาเป็นกันเอง
- มีมุกตลกนิดหน่อย
- มีคำขอบคุณ
- มีคำอวยพร
```

## ⚠️ **ข้อผิดพลาดที่พบบ่อย**

### **1. Prompt สั้นเกินไป**

```
❌ "เขียนโพสต์"
✅ "เขียนโพสต์ Facebook แจ้งเตือนชาวบ้านเรื่อง..."
```

**แก้:** ให้ข้อมูลครบถ้วน

---

### **2. ไม่กำหนดบทบาท**

```
❌ "เขียนโครงการ"
✅ "คุณคือผู้เชี่ยวชาญการเขียนโครงการ จงเขียน..."
```

**แก้:** กำหนดบทบาทให้ AI

---

### **3. ไม่กำหนดรูปแบบ**

```
❌ "เขียนสรุป"
✅ "เขียนสรุป ความยาว 200 คำ เป็นข้อๆ"
```

**แก้:** กำหนดรูปแบบที่ต้องการ

---

### **4. ข้อมูลไม่ครบ**

```
❌ "เขียนประกาศประชุม"
✅ "เขียนประกาศประชุม วันเสาร์ที่ 13 เม.ย. 69 เวลา 10:00 น. ณ..."
```

**แก้:** ให้ข้อมูลครบถ้วน

---

### **5. ไม่ตรวจสอบผลลัพธ์**

```
❌ ได้ผลลัพธ์มา → ใช้เลย
✅ ได้ผลลัพธ์มา → ตรวจสอบ → แก้ไข → ใช้
```

**แก้:** ตรวจสอบก่อนใช้เสมอ

## ✍️ **แบบฝึกหัด**

### **แบบฝึกหัดที่ 1: เขียนประกาศ**

**โจทย์:** เขียนประกาศรับสมัครอาสาสมัครพัฒนาชุมชน

**ข้อมูล:**
