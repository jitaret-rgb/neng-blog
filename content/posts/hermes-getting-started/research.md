# Research: Hermes Agent Installation & Usage

## Key Facts
1. Hermes Agent is an open-source CLI AI assistant built by Nous Research
2. Installation: `curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash`
3. Manual install requires Python 3.11+, uv, and git clone
4. Config file at `~/.hermes/config.yaml`
5. Supports multiple providers: OpenRouter, Kimi, Bailian, OpenAI, etc.
6. Has gateway integrations: Telegram, Discord, WhatsApp, Slack, Signal
7. Supports MCP (Model Context Protocol) servers like Lightpanda
8. Features skills system, memory system, cron jobs, subagent delegation
9. OpenClaw is a fork/related project that some users migrate from
10. Hermes uses toolsets: terminal, file, web, browser, vision, etc.
